# Portfolio Website - Mochammad Rezky Ramadhan

## Tampilan Section
1. Home (Hero Section)
   - Foto profile
   - Nama dan deskripsi
   - Tombol Instagram & LinkedIn

2. About Me
   - Deskripsi diri
   - Skills dengan progress bar
   - Pengalaman organisasi dan project

3. Certificates

## Penjelasan Code
- Menggunakan Bootstrap 5 untuk navbar, grid system, card, button, dan progress bar.
- Menggunakan CSS untuk custom styling seperti gradient background dan foto bulat.
- Website bersifat statis dan responsive.

## Teknologi yang Digunakan
